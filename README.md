Employee Management System (React + Vite)

A modern Employee Management System (EMS) built using React and Vite, designed to manage employee records efficiently with full CRUD functionality.

🚀 Features
➕ Add new employees
📋 View employee list
✏️ Edit employee details
❌ Delete employees
🔍 Search and filter employees (optional)
⚡ Fast performance with Vite
♻️ Hot Module Replacement (HMR)
🛠️ Tech Stack
Frontend: React
Build Tool: Vite
Styling: CSS / Bootstrap (optional)
API: JSON Server / REST API
Linting: ESLint
